# SecuTrans - 文件安全传输工具

## 📋 项目简介

SecuTrans 是一个基于网络的文件安全传输工具，实现了发送方和接收方对文件的加密发送和解密接收功能。项目采用 Python + Tkinter 开发，界面采用桂电蓝色系设计。

## ✨ 主要特性

### 🔐 安全功能
- **现代对称加密算法**：AES-128/192/256、ChaCha20、SM4、Camellia
- **传统算法支持**：DES、3DES、Blowfish、RC4（标记为不安全，仅作演示）
- **多种加密模式**：ECB、CBC、GCM、CFB、OFB、CTR
- **安全标记系统**：明确标识不安全的算法和模式
- **数字信封技术**：保障文件传输过程中的安全性
- **RSA公钥加密**：保护对称密钥的安全传输
- **数据完整性验证**：通过SHA-256哈希验证文件完整性

### 🖥️ 用户界面
- **统一界面设计**：发送和接收功能合并到单一界面
- **四层布局结构**：密钥管理、文件管理、文件传输、日志记录
- **系统原生风格**：使用标准按钮，保持操作系统原生风格
- **完整菜单系统**：文件菜单、工具菜单、帮助菜单
- **快捷键支持**：常用操作快捷键（Ctrl+O、Ctrl+G、F2、F3等）
- **多颜色日志**：普通黑色、成功绿色、警告黄色、错误红色
- **实时进度条**：发送和接收都有独立进度显示
- **自动配置保存**：所有设置自动保存，下次启动自动加载
- **文件信息完整**：显示文件名、类型、大小、修改时间、路径
- **安全状态指示**：算法和模式的安全状态实时显示
- **端口标准化**：默认使用5375端口（SeTs）

### 🌐 网络功能
- **本地网络传输**：支持局域网内文件传输
- **实时连接状态**：显示连接信息和传输状态
- **多线程处理**：避免界面卡顿，提升用户体验

## 🚀 快速开始

### 环境要求
- Python 3.7+
- tkinter (通常随Python安装)
- 所需依赖包见 requirements.txt

### 安装步骤

1. **克隆项目**
```bash
git clone <repository-url>
cd SecuTrans
```

2. **安装依赖**
```bash
pip install -r requirements.txt
```

3. **运行程序**
```bash
python main.py
```

或者使用启动器：
```bash
python tests/start.py
```

## 📖 使用说明

### 基本使用流程

#### 1. 准备密钥对
- 在"发送文件"标签页点击"生成测试密钥对"按钮
- 将公钥文件发送给接收方
- 接收方在"接收文件"标签页加载对应的私钥文件

#### 2. 发送文件
- 切换到"发送文件"标签页
- 选择要发送的文件
- 选择加密算法和模式
- 加载接收方公钥
- 连接到接收方服务器
- 点击"发送文件"

#### 3. 接收文件
- 切换到"接收文件"标签页
- 启动服务器
- 加载私钥文件
- 选择文件保存路径
- 等待发送方连接和文件传输

### 高级功能

#### 加密算法选择
- **AES-128/192/256**：高级加密标准，推荐使用（✓安全）
- **ChaCha20**：现代流加密算法，高性能（✓安全）
- **SM4**：中国国家标准的分组密码算法（✓安全）
- **Camellia**：日本开发的分组密码算法（✓安全）
- **DES**：数据加密标准，已不安全（⚠不安全，仅作演示）
- **3DES**：三重DES，已不安全（⚠不安全，仅作演示）
- **Blowfish**：对称块加密，已不安全（⚠不安全，仅作演示）
- **RC4**：流加密算法，已不安全（⚠不安全，仅作演示）

#### 加密模式选择
- **GCM**：认证加密模式，提供机密性和完整性（✓推荐）
- **CBC**：密码块链接模式（常用）
- **CFB**：密码反馈模式
- **OFB**：输出反馈模式
- **CTR**：计数器模式
- **ECB**：电子密码本模式（⚠不安全，仅作演示）

## 📝 注意事项

1. **防火墙设置**：确保选定的端口未被防火墙阻止
2. **网络连接**：发送方和接收方需要在同一网络中
3. **文件大小**：默认最大支持100MB文件传输
4. **密钥安全**：请妥善保管私钥文件，避免泄露

## 🐛 常见问题

### Q: 连接失败怎么办？
A: 检查IP地址和端口号是否正确，确保防火墙允许连接。

### Q: 解密失败怎么处理？
A: 确认使用的是正确的私钥文件，检查文件是否在传输过程中损坏。

### Q: 支持哪些文件格式？
A: 支持所有格式的文件，传输过程采用二进制模式。
