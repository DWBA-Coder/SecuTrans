# SecuTrans - 文件安全传输工具

## 📋 项目简介

SecuTrans 是一个基于网络的文件安全传输工具，实现了发送方和接收方对文件的加密发送和解密接收功能。项目采用 Python + Tkinter 开发，界面采用桂电蓝色系设计。

## ✨ 主要特性

### 🔐 安全功能
- **多种对称加密算法支持**：AES、DES、3DES、Blowfish、RC4
- **多种加密模式选择**：ECB、CBC、CFB、OFB、CTR
- **数字信封技术**：保障文件传输过程中的安全性
- **RSA公钥加密**：保护对称密钥的安全传输
- **数据完整性验证**：通过SHA-256哈希验证文件完整性

### 🖥️ 用户界面
- **现代化Tkinter界面**：采用桂电蓝(#005FA5)主题色
- **双窗口设计**：独立的发送方和接收方窗口
- **实时进度显示**：传输进度条和详细日志
- **友好操作提示**：清晰的状态信息和错误提示

### 🌐 网络功能
- **本地网络传输**：支持局域网内文件传输
- **实时连接状态**：显示连接信息和传输状态
- **多线程处理**：避免界面卡顿，提升用户体验

## 🚀 快速开始

### 环境要求
- Python 3.7+
- tkinter (通常随Python安装)
- 所需依赖包见 requirements.txt

### 安装步骤

1. **克隆项目**
```bash
git clone <repository-url>

```

2. **安装依赖**
```bash
pip install -r requirements.txt
```

3. **运行程序**
```bash
python secutrans.py
```

## 📖 使用说明

### 基本使用流程

#### 1. 准备密钥对
- 在发送方点击"生成测试密钥对"按钮
- 将公钥文件发送给接收方
- 接收方加载对应的私钥文件

#### 2. 发送文件
- 打开"文件发送方"窗口
- 选择要发送的文件
- 选择加密算法和模式
- 加载接收方公钥
- 连接到接收方服务器
- 点击"发送文件"

#### 3. 接收文件
- 打开"文件接收方"窗口
- 启动服务器
- 加载私钥文件
- 选择文件保存路径
- 等待发送方连接和文件传输

### 高级功能

#### 加密算法选择
- **AES**：高级加密标准，支持128/192/256位密钥
- **DES**：数据加密标准，56位密钥
- **3DES**：三重DES，112/168位密钥
- **Blowfish**：对称块加密，可变密钥长度
- **RC4**：流加密算法，可变密钥长度

#### 加密模式选择
- **ECB**：电子密码本模式
- **CBC**：密码块链接模式
- **CFB**：密码反馈模式
- **OFB**：输出反馈模式
- **CTR**：计数器模式

## 📝 注意事项

1. **防火墙设置**：确保选定的端口未被防火墙阻止
2. **网络连接**：发送方和接收方需要在同一网络中
3. **文件大小**：默认最大支持100MB文件传输
4. **密钥安全**：请妥善保管私钥文件，避免泄露

## 🐛 常见问题

### Q: 连接失败怎么办？
A: 检查IP地址和端口号是否正确，确保防火墙允许连接。

### Q: 解密失败怎么处理？
A: 确认使用的是正确的私钥文件，检查文件是否在传输过程中损坏。

### Q: 支持哪些文件格式？
A: 支持所有格式的文件，传输过程采用二进制模式。
